# Daily Learning Tracker 🚀

Welcome to my daily learning repository! This project serves as a structured space to document my learning journey, store coding solutions, write down notes, and track progress across core areas of computer science and software development.

## 📂 Repository Structure

The repository is organized into three main learning modules:

*   **[DSA/](./DSA)**: Data Structures and Algorithms exercises, patterns, and complexity analysis.
*   **[Python/](./Python)**: Python programming concepts, scripts, frameworks, and utility tools.
*   **[SQL/](./SQL)**: Database queries, design, optimizations, and schema challenges.

---

## 🧠 Data Structures & Algorithms (DSA)
In this section, I focus on problem-solving techniques and algorithmic efficiency (Big O notation).
*   **Topics:** Arrays, Linked Lists, Trees, Graphs, Dynamic Programming, Sorting & Searching.
*   **Goal:** Solve and document daily coding challenges (e.g., LeetCode, HackerRank).

## 🐍 Python Programming
This folder contains language-specific explorations, scripts, and implementation of Pythonic design patterns.
*   **Topics:** Object-Oriented Programming (OOP), Functional Programming, Decorators, Generators, Concurrency, and libraries.
*   **Goal:** Build practical scripts and explore advanced features of the Python language.

## 🗄️ SQL & Databases
Here I document database query optimizations, normalization, and relational schema designs.
*   **Topics:** Joins, Subqueries, CTEs, Window Functions, Indexing, and Transaction management.
*   **Goal:** Master relational database management systems (RDBMS) such as PostgreSQL, MySQL, and SQLite.
